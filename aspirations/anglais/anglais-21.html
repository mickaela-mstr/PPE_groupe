    Here is one work with legend text getting cut off or when you have long legend values:

    1. Turn off legends on the various visuals you want to display

    2. Create a Treemap visual with same data columns. Place its legend on "Top Left ". Reduce height of the Tree Map visual so that only the Legend displays. I set it to 60px and only the legend was showing. I set the width of the tree map visual as wide as possible to accomodate long values in my Legend's data category.

     

    Hope this helps.
