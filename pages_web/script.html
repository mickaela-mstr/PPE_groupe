Scripts Bash - Explications

run_all.sh

exécute aspiration_all.sh, dump_all.sh, contextes_all.sh, concordancier.sh, tableau_all.sh, vérifie leur existence, leur attribue les permissions d'exécution et arrête l'exécution en cas d'erreur.

aspiration_all.sh

Ce script parcourt une liste d'URL pour chaque langue (anglais, français, arabe) et télécharge les pages web correspondantes. Les pages sont enregistrées au format HTML avec des informations complémentaires.

dump_all.sh

Utilise lynx pour extraire le texte brut de chaque page web aspirée. Ces textes sont sauvegardés dans le dossier dumps-text.

contextes_all.sh

Identifie et extrait les contextes pertinents pour un mot-clé donné. Les phrases contextuelles sont sauvegardées pour chaque langue dans un fichier séparé.

concordancier.sh

Ce script génère des fichiers HTML pour visualiser les concordances des mots-clés. Les résultats sont formatés sous forme de tableaux clairs et lisibles.

tableau_all.sh

Crée des tableaux synthétiques regroupant les métadonnées importantes de chaque page aspirée : URL, statut HTTP, encodage, liens, etc.

pals.sh

prend en entrée un dossier contenant des fichiers .txt (soit dumps-text soit contextes), traite chaque fichier en séparant les mots par des sauts de ligne, puis sauvegarde le résultat dans un dossier de destination (pals). Il vérifie également l'existence du dossier et des fichiers avant de procéder au traitement.

make_pals_corpus.sh

prend un fichier de texte par langue (arabe, français ou anglais), transforme son contenu en séparant les mots ou les phrases, puis enregistre ce contenu transformé dans un fichier de sortie pour une analyse ou un traitement ultérieur.
